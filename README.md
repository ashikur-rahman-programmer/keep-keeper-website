# KeenKeeper - Keep Your Friendships Alive

## Project Description

**KeenKeeper** is a modern friendship management web application that helps users stay connected with friends.
Users can track interactions, monitor relationship goals, check friend details and maintain meaningful connections through calls, texts and video check-ins.

## Technology Used

- Next.js
- Tailwind CSS + DaisyUI
- Recharts
- React Toastify
- Context API
- Custom Hook
- Local JSON Data (SSR with fs)
- Vercel Deployment

---

## Key Features

### Friends Management

- Display friends from JSON data
- Detailed friend profile page
- Status tracking (Overdue / Almost Due / On Track)

### Timeline System

- Add Call, Text, Video interactions
- Auto-generated timeline entries with date
- Toast notification on interaction
- Filter and Search functionality

### Stats Analytics

- Pie chart showing interaction types
- Dynamic data based on timeline
- Tooltip and Legend added
- Empty state UI when no data
